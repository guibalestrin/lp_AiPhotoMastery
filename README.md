# AI Photo Mastery

Landing page em React + Vite para a marca Caverna de Alexandria.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
npm run typecheck
```

## Deploy na Vercel

- Framework preset: `Vite`
- Install command: `npm install`
- Build command: `npm run build`
- Output directory: `dist`

## Status de validacao

- `npm run build` OK
- `npm run lint` OK
- `npm run typecheck` OK

## Observacoes

- O projeto nao depende de variaveis de ambiente para renderizar a landing page.
- A secao hero usa um asset local do projeto para evitar quebra por URL externa.
